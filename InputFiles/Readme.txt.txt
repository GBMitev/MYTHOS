16O-1H__MYTHOS.inp contains the 4 state model used to produce the MYTHOS line list
	Bound state PECs - X, A, B, C
	Coupling Curves  - SOCs, EAMCs, Spin-Rot, Lambda-Doubling
	(T)DMCS          - X-ABC, A-A, B-B, C-C

16O-1H__MYTHOS_Full.inp contains all information:
	Bound state PECs - X, A, B, C
	Repulsive States - 12Sigma-,14Sigma+,14Pi,12Delta
	Coupling Curves  - SOCs, EAMCs, Spin-Rot, Lambda-Doubling
	(T)DMCS          - X-ABC, A-A, B-B, C-C, X-12Sigma-, X-12Delta

16O-1H__MYTHOS.pf is the partition function dependent on temperature
